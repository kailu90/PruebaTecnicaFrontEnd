# Juego de Parejas con Tarjetas.

## Descripción del Proyecto
El objetivo del juego es encontrar todos los pares de cartas coincidentes en el menor número de movimientos posible.
El proyecto está desarrollado bajo principios los lenguajes de HTML, CSS y JS vanilla, sin frameworks.

### Características Clave:
**Diseño Responsivo:** Jugable en cualquier dispositivo (móvil, tablet, escritorio).
**Contador de Movimientos:** Seguimiento en tiempo real de la puntuación del usuario.
**Contador de Aciertos:** Seguimiento en tiempo real de los aciertos obtenidos por el jugador.
**Temporizador:** Temporizador toma los segundos/minutos desde que se inicia el juego.
**Lógica Clara:** Manejo de estado eficiente para voltear, emparejar y resetear el tablero.
**Optimización de UX:** Uso de transiciones CSS para un *feedback* visual suave y agradable.


### Tecnologías Utilizadas
**Estructura** 
HTML 5, Estructura semántica del juego.
**Estilos** 
CSS3, Estilos y animaciones aplicados directamente con CSS.
**Lenguaje**
JS (ES6+), Toda la lógica de la aplicación (interactividad, estado del juego, temporizador).
**Control de Versiones** 
Git/GitHub Para el control de versiones del código fuente.
